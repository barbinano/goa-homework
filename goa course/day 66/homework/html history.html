HTML-ის შექმნა

HTML შეიქმნა 1991 წელს ტიმ ბერნერს-ლის მიერ. მისი მიზანი იყო ვებსაიტებზე ინფორმაციის სტრუქტურირებული წარმოდგენა.

HTML-ის განვითარება

დროთა განმავლობაში HTML განვითარდა: HTML2, HTML4 და საბოლოოდ HTML5, რომელმაც დაამატა ვიდეო, აუდიო და სემანტიკური ელემენტები.

HTML5 ოფიციალურად გამოვიდა 2014 წელს და გახდა თანამედროვე ვებ-დეველოპმენტის საფუძველი.

ეს ტექსტი იყენებს inline style-ს, რაც ყველაზე მაღალი სპეციფიკურობისაა.
